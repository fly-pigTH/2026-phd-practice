任务描述
本项目面向异构智算基础设施快速发展背景下算力利用率不足的问题，针对当前大模型训练过程中在 Kernel 优化、分布式训练策略以及集群资源调度等多个层面存在的效率瓶颈与适配困难，系统开展多智能体驱动的全栈效能优化研究。具体而言，在 GPU Kernel 层，现有优化方法高度依赖人工经验且难以跨架构迁移；在训练任务层，并行策略对硬件环境依赖性强，跨平台复用能力不足；在集群层，多用户并发环境下资源分配与调度机制不完善，导致算力利用率低与服务稳定性不足。与此同时，大语言模型与多智能体技术的发展为构建“分析、决策、执行、反馈”的闭环优化体系提供了新机遇。本项目旨在通过引入多智能体协同推理与自适应演化机制，打通 Kernel、训练与集群三个层级之间的协同优化路径，实现面向异构环境的高效、稳定与可迁移的算力优化新范式。该项目在大模型训练推理加速、高效能训练、以及智能算力平台适配等场景中具有广阔的应用前景，可为智能计算系统优化和新型算力基础设施建设提供技术支撑。
面向异构智算集群的多智能体驱动全栈训练效能优化实践：旨在构建一个多智能体驱动的全栈算力效能优化系统雏形，在6周实践周期内实现覆盖 GPU Kernel 优化、分布式训练策略优化以及集群算力调度优化的跨层闭环原型。同时输出完整的技术报告及具备投稿潜力的论文初稿，从而验证多智能体方法在全栈算力优化中的可行性与应用潜力。
1. 需要解决多智能体协同驱动的全栈优化机制设计问题，构建统一框架打通 GPU Kernel 优化、分布式训练策略优化与算力调度优化，设计性能分析、策略生成与反馈评估等智能体协同流程，实现跨层联动与闭环优化 2. 需要解决面向异构平台的适配与迁移优化问题。针对不同 GPU 架构、通信机制及软件栈差异带来的性能不稳定与优化不可复用问题，需构建统一的抽象表示，实现 Kernel 优化策略、训练配置及调度策略在不同硬件环境下的自适应迁移与快速调优，提升系统的泛化能力与工程可落地性。
目前，已初步完成 GPU Kernel 优化、训练策略优化的多智能体系统的搭建，并已取得较好的实验效果，具备一定的前期研究基础。同时，团队已积累了相关代码样本、性能数据和优化实验结果，可为后续数据集构建、模型训练和系统优化提供支撑。实验室拥有充足的计算资源和软硬件环境，可满足深度强化学习训练、CUDA Kernel 测试及大规模实验评测需求，并具备较完善的人力资源支持，能够保障项目顺利实施
