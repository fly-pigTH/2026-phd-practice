任务描述
当前，政府部门虽然掌握了大量数据，但分散在不同系统，缺乏统一的数据底座和可视化工具，导致产业监测“看不清”、企业服务“摸不准”、领导决策“缺抓手”。同时，面对每日大量的行业动态和政策更新，人工监测效率低、响应慢。本项目旨在基于开源框架打造“行业资讯”、“精准招商”等数字员工，实现行业动态自动采集、政策精准推送、招商线索智能挖掘，并且通过数据集成与可视化技术，将零散数据转化为动态产业地图与企业画像，推动产业治理从“经验驱动”迈向“智能体驱动”。
1.智能体数字员工：基于开源框架构建具备行业资讯采集、政策知识库检索（RAG）、招商线索识别、定时简报生成与多渠道推送能力的数字员工，实现产业服务“无人值守、智能响应”。 2.数据可视化界面：完成多源数据集成治理，开发产业地图（GIS）、企业画像、预警面板等可视化模块。
1.让智能体具备自主完成信息采集、知识检索、内容生成与推送的系列能力，并通过自动化机制实现日常资讯的定时整理与多平台分发。2.把来自不同部门、格式各异的数据有效整合起来，形成统一、可信的企业信息基础。
提供项目所需数据资料和服务器等软硬件设备，提供业务指导人员和技术配合团队。
第1周：完成开发环境部署及数字员工场景设计。第2周：完成 MCP Server 构建及基础对话能力，完成政策知识库构建及资讯采集 Skill。第3周：完成报告生成 Skill 及定时任务配置，产业地图展示页面初步完成。第4周：画像与预警模块开发；性能优化与画像交互完善。第5周：UI 打磨与演示准备，完成多渠道推送及对话体验优化。第6周：系统整合演示，交付项目文档与架构说明。
